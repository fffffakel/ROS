===
Название: pwd
Описание: Показывает текущий рабочий каталог
Синтаксис: pwd [опции]
Пример: pwd
Примечание Полезна для определения, в каком каталоге находишься

===
Название: ls
Описание: Показывает содержимое каталога
Синтаксис: ls [опции] [каталог]
Пример: ls
Примечание:

===
Название: cd
Описание: Переходит в указанный каталог
Синтаксис: cd [путь_к_каталогу]
Пример: cd /home/user
Примечание: Пустой аргумент возвращает в домашний каталог

===
Название: mkdir
Описание: Создаёт новый каталог
Синтаксис: mkdir [опции] [имя_каталога]
Пример: mkdir newfolder
Примечание: Создаёт каталог с указанным именем в текущей папке

===
Название: rmdir
Описание: Удаляет пустой каталог
Синтаксис: rmdir [опции] [имя_каталога]
Пример: rmdir oldfolder
Примечание: Удаляет каталог только если он пустой

===
Название: ps
Описание: Показывает список текущих процессов пользователя или всей системы
Синтаксис: ps [опции]
Пример: ps aux
Примечание: aux — подробный список всех процессов
===
Название: cat
Описание: Выводит содержимое файла в терминал
Синтаксис: cat [опции] [имя_файла]
Пример: cat myfile.txt
Примечание: Можно объединять несколько файлов в вывод (cat file1 file2)

===
Название: cp
Описание: Копирует файлы или каталоги
Синтаксис: cp [опции] [исходный_файл] [целевой_путь]
Пример: cp file.txt backup.txt
Примечание:

===
Название: mv
Описание: Перемещает или переименовывает файлы и каталоги
Синтаксис: mv [опции] [исходный_файл] [новый_путь]
Пример: mv oldname.txt newname.txt
Примечание: Можно использовать для перемещения в другой каталог

===
Название: rm
Описание: Удаляет файлы или каталоги
Синтаксис: rm [опции] [имя_файла]
Пример: rm file.txt
Примечание: Для удаления каталога с содержимым использовать rm -r
